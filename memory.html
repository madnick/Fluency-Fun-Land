Memory Game

Memory game exercises offer you and your child a fun way to work through memory and visual recognition skills. Memory games enable a child to focus on image details and associate relationships between matching pairs.

Instructions

Playing the memory game involves using your memory skills to match the card pairs. Click two cards in attempt to match the pairs. If the child clicks on two cards that do not match, they will be displayed and flipped back. Successful matches will disappear from the selection.
